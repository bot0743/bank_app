Как поднять сайт банка?

1. Открыть терминал слева снизу / Alt+F12
2. Прописать команды 
2.1. pip install -r requirements.txt для установки требований.
2.2. После установки требований пишем следующую команду - flask init-db для установки и инициализации базы данных и после flask run
2.3. Чтобы Банк находил вас по логину и почте нужно также установить email_validator по команде pip install email_validator
После всех установок - можете запускать сайт и регистрироваться, ваш логин, почта и пароль будет хранится в базе данных и к ней можно будет обращаться, - перевод или запрос перевода
Регистрируетесь и заходите в свой профиль



set FLASK_APP=app.py
